## Local development
- Create a `.env` file in the root folder of the repo with the below contents:
    ```dotenv
    VITE_MAPBOX_TOKEN=insert_mapbox_token_here
    ```
- Run the below commands
    ```bash
    yarn install
    yarn start
    ```
